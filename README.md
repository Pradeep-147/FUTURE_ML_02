# FUTURE_ML_02

Customer Churn Prediction

Overview
End-to-end machine learning project to predict customer churn and identify key drivers affecting customer retention.

Key Features
Data cleaning and feature engineering
Models: Logistic Regression, Random Forest, XGBoost
Evaluation using accuracy, precision, and recall
Churn probability prediction
Business insights and recommendations

Tools
Python, Pandas, Scikit-learn, XGBoost, Matplotlib

Outcome
Identified high-risk customers and major churn drivers such as contract type, tenure, and monthly charges.
